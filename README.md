# RedMCacheCleaner
This script automatically removes all cache folders (cache, server-cache, server-cache-privat) from the directory: C:\Users\vedsc\AppData\Local\RedM\RedM.app\data\

Important: 
- The path must be adjusted accordingly. Instead of vedsc, replace it with your own username.
- This ensures that outdated or corrupted data is deleted, helping to avoid potential issues with RedM.


Usage:
- Download the script and make sure you have the necessary permissions to delete these files.
- Use Win-PS2EXE to convert the PowerShell script into an executable .exe file. You can download the program here: [Win-PS2EXE.](https://github.com/MScholtes/Win-PS2EXE)
- Run the generated .exe as administrator, as the script will not work otherwise.

Warning
⚠️ Caution: The script permanently deletes files. Make sure you are deleting the correct folders and have backed up any important data.
